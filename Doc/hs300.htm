Start FreeCAD

Open the CodeSharkFCs application
Select the FreeCAD menu item




Select Run FreeCAD from the sub menu.



The FreeCAD Interface dialog is displayed.



Click the Run FreeCAD button.



The FreeCAD application starts with a blank project.  The panels that are visible on inital startup are specified by the script PanelViewScript.py.



Use the FreeCAD File Menu item to open the desired drawing file.



The drawing file is displayed.
