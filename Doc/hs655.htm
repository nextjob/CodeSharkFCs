HighLighter



G Code Highlighting in the editor window is controlled by this dialog.  The text entered into the 'G Code String' text boxes will be highlighted to the color selected in the 'Highlight Color' dropdown list on the editor window.  Highlighting is performed as a simple text pattern match, except for the form "G Code Address"# (ie T#).  When the G Code address value is postfixed by the '#" symbol, all numberic characters following the G Code address will highlighted.
